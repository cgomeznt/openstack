# OpenStack

Nada de lo que esta aqui intenta sustituir la documentación oficial de OpenStack
http://docs.openstack.org/juno/install-guide/install/apt/content/

La meta de estos scripts es ayudar a terminar de interpretar con la practica la información suministrada por Openstack....!!!

Estaremos documentando los pasos para su buen uso y en la espera que nos ayuden a mejorar los script para mejorar el aprendizaje de OpenStack.
